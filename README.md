# simple-vue
一个简单的vue实现：目前实现了vue的双向绑定，包括对象、数组的绑定，持续更新中
该项目实现了 v-model v-bind 的双向绑定的指令；支持双括号语法



## 下载

```
git clone git@github.com:Kryst4ljy/simple-vue.git
```

## 使用

```
npm i
npm run dev
```
